# Taskly API

A Django REST Framework API for managing household tasks.

## Features
- User authentication
- Task & TaskList management
- Role-based access
- Token-based OAuth2 authentication

## Tech Stack
- Python, Django, DRF
- PostgreSQL
- Render (for deployment)
